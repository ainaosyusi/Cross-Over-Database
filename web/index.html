    CrossOverデータベースについて

    このツールは軽音サークルのYouTubeプレイリストにある動画の概要欄を解析し、演奏統計を自動集計しています。

    全58プレイリストを解析対象としています。

    データ抽出方法
    YouTube動画の概要欄からプログラムで自動抽出しています。手動入力ではないため、誤りがある場合があります。
    表記揺れの統合
    同一人物の表記揺れ（旧字体、ニックネーム等）は可能な限り統合しています。
    ジャンル分類
    アーティストのジャンル分けはすべてAIによる自動分類です。明らかに違うものがありますが、気が向いたら修正します。
    プライバシー
    このページはメンバー専用です。
    技術構成
    Python + JavaScript (static site)

    一部のイベントはデータが不完全な場合があります。情報が判明次第、順次更新しています。

    情報をお持ちの方は、ぜひご提供ください！より正確な統計を目指しています。

    情報の誤りや追加データがある方は管理者にご連絡ください。

    ⚠️このサイトをブックマークしよう

    このサイトは検索エンジンには表示されません。ブックマークまたはホーム画面への追加をしないとアクセスできなくなります。

    サイトの後継管理者がいない場合でも、最低でもあと1年はこのサイトは運用を続けます。それまでデータの閲覧・更新が可能です。

    ランキング機能ガイド

    バンド数 / 曲数
    出演バンド数・演奏曲数の多い人ランキング。
    多様性
    何種類のアーティストの曲を演奏したか。幅広いジャンルに挑戦している人が上位に。
    人気曲 / 人気アーティスト
    演奏回数の多い曲・アーティスト。タップで詳細表示。
    ベストコンビ
    同じバンドで共演した回数が多いペア。
    視聴回数 / 視聴回数メンバー
    YouTube動画の再生回数が多い演奏と、その出演者の累計視聴回数。
    トリ率
    イベントの最後のバンド（トリ）を務めた割合。
    交流度
    共演したユニークな人数。多くの人とバンドを組んでいる人が上位に。
    パート別
    パートごとの出演回数と担当メンバー一覧。タップで展開。
    名前を入力して検索してください
